## Plasma Component

When using the Plasma WebGL component in this project:

1. **Always import from** `/src/components/Plasma.jsx` (not `/ui/Plasma.tsx`)
   - `/components/Plasma.jsx` = OGL WebGL library (correct)
   - `/components/ui/Plasma.tsx` = CSS gradient mock (wrong)

2. **Supported props** for OGL Plasma:
   - `color` - hex color string (#RRGGBB)
   - `speed` - animation speed multiplier (0-2)
   - `direction` - "forward", "backward", or "pingpong"
   - `scale` - scale factor (0.5-2)
   - `opacity` - 0-1 alpha value
   - `mouseInteractive` - boolean for mouse tracking

3. **Container sizing** - Always use full-screen or explicit dimensions in the parent div

If the Plasma effect doesn't appear at all, check:
- ✓ Correct import path (Plasma.jsx, not ui/Plasma.tsx)
- ✓ Container has dimensions (width/height)
- ✓ z-index layering if behind other elements
